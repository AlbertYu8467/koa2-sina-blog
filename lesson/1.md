# ejs 服务端模板引擎
1. 变量
`<%= title %>`
`<%= locals.name %>` 
2. 判断
```
  <% if(isMe){ %>
    <a href="#">@提到我的(3)</a>
  <% } else { %>
    <button>关注</button>
  <% } %>
```
3. 循环
```
<ul>
<% blogList.forEach(blog => { %>
  <li><%= blog.title %></li>
<% }) %>
</ul>
```
4. 引入组件
```
    <%- include('widgets/user-info', {
      isMe
    })%>
``
